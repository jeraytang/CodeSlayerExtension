# Code Slayer

## 项目背景

摸鱼无聊？用 Trae CN 写一个 VS Code 插件

## 功能特性

- **敌人出现**：写代码时，每行代码会产生一个敌人
- **快捷键斩杀**：按 `Ctrl+S` 保存文件并斩杀敌人
- **连击系统**：连续斩杀敌人会增加连击数，连击越高敌人越多
- **Boss系统**：代码量达到阈值时会出现Boss
- **音效系统**：敌人出现、击杀、Boss出现和Boss击杀都有相应的音效
- **页面切换**：切换代码文件时，敌人状态会重新开始
- **成就系统**：完成特定任务解锁成就
- **高级特效**：零错误保存时会触发震撼的刀光碎裂特效
- **多种模式**：支持三种不同的怪物生成模式

## 安装方法

1. 克隆或下载本项目
2. 打开 VS Code
3. 点击左侧的扩展图标（方块图标）
4. 点击右上角的三个点，选择 "从 VSIX 安装..."
5. 选择编译后的 `.vsix` 文件安装

## 编译项目

1. 安装依赖：
   ```bash
   npm install
   ```

2. 编译项目：
   ```bash
   npm run compile
   ```

3. 监视模式（开发时使用）：
   ```bash
   npm run watch
   ```

## 使用说明

1. **启动游戏**：按下 `Ctrl+Shift+P` 打开命令面板，输入 "Code Slayer: 开始游戏" 并执行
2. **写代码**：开始编写代码，敌人会根据你选择的模式出现
3. **斩杀敌人**：按 `Ctrl+S` 保存文件并斩杀敌人
4. **停止游戏**：按下 `Ctrl+Shift+P` 打开命令面板，输入 "Code Slayer: 停止游戏" 并执行

## 游戏模式

Code Slayer 支持三种不同的怪物生成模式，你可以根据自己的喜好选择：

### 1. onTyping 模式（默认）
- **触发条件**：写代码换行时
- **特点**：每行代码生成一个敌人
- **适合**：喜欢边写边杀的玩家

### 2. onError 模式
- **触发条件**：代码出现诊断错误时
- **特点**：
  - 语法错误 → 普通小怪
  - 警告 → 带护甲的怪
  - 类型错误 → 魔法怪
  - ESLint 问题 → 粘液怪（减速）
  - 修复错误 → 太刀一闪斩杀
- **适合**：喜欢挑战的玩家，鼓励写出无错误的代码

### 3. hybrid 模式
- **触发条件**：结合以上两种模式
- **特点**：小怪按换行，精英怪/Boss 按错误
- **适合**：想要平衡体验的玩家

## 如何切换模式

1. **打开设置**：点击 VS Code 左下角的齿轮图标，选择 "设置"
2. **搜索配置**：在搜索框中输入 "Code Slayer"
3. **选择模式**：在 "Code Slayer: Monster Spawn Mode" 选项中选择你喜欢的模式

## 高级配置

除了模式切换，你还可以调整以下配置：

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `Code Slayer: Monster Spawn Mode` | `onTyping` | 怪物生成模式 |
| `Code Slayer: Max Monsters` | `30` | 最大怪物数量 |
| `Code Slayer: Error Detection Delay` | `500` | 错误检测延迟(ms) |

## 游戏机制

- **敌人出现**：根据选择的模式生成敌人
- **Boss出现**：当代码量达到30行时，会出现Boss
- **连击系统**：连续斩杀敌人会增加连击数，连击越高，敌人出现的速度越快
- **Perfect Save**：零错误保存时会触发刀光碎裂特效和全屏清场
- **成就系统**：
  - 初出茅庐：达到5连击
  - 连击达人：达到20连击
  - Boss猎人：击杀Boss
  - 代码狂人：写1000行代码
